# QQ Chat Box

一个基于 Python 的 QQ 风格聊天气泡生成工具，可以自动获取 QQ 用户信息并生成聊天截图。将你在文本输入框中输入的文字，转变为选定qq的聊天气泡。它能够创建出与qq聊天记录类似的效果，适用于QQ

**重要声明：本工具仅限于实验教学用途，请勿用于任何非法或侵权用途。使用者应对其行为承担全部法律责任，开发者不承担任何因使用本工具而产生的法律责任。**

## 基础命令
- ctrl+1 : 输入qq号
- ctrl+2 : 选择头衔和填写头衔信息
- ctrl+3 : 使用备注
